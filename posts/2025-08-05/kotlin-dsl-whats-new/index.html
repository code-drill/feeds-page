Kotlin DSL: What’s New?

2025-08-05 13:57

🚀 TeamCity has introduced significant enhancements to the Kotlin DSL in recent releases! 📁 Users can now store Kotlin DSL files outside the .teamcity directory, improving visibility and convenience when working with IntelliJ IDEA. 🔄 Additionally, code reuse is simplified with the introduction of DSL libraries, allowing developers to share code across projects more efficiently. 📊 For those managing multiple branches, changes in DSL files can now better align with build configurations.
