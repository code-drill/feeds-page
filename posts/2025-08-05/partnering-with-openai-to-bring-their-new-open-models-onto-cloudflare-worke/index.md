🚀 Exciting news! OpenAI's latest open-source models are now live on Cloudflare Workers AI. Developers can access two new models: a 120 billion parameter and a 20 billion parameter version. These models feature a Mixture-of-Experts architecture for efficient performance and are designed for transparency and flexibility. 🛠️ In addition, Cloudflare supports the new Responses API, enhancing interaction capabilities. The Code Interpreter and Web Search features are also on the way! #OpenAI...
