🚀 Microfrontends support is now in Public Beta! This feature allows developers to split large applications into smaller, manageable parts, enabling faster development and independent testing. Vercel simplifies the assembly and routing of these microfrontends into a cohesive user experience. To get started, users can clone examples from the Vercel dashboard, create microfrontend groups, and configure their projects with ease. For more details, check the documentation or contact Vercel support....
