Managing user access for Red Hat Enterprise Linux (RHEL) just became simpler! 🚀 Red Hat has introduced three new system roles in the User Access service of the Hybrid Cloud Console: 1️⃣ **RHEL Administrator**: Full privileges for managing configurations and vulnerabilities. 2️⃣ **RHEL Operator**: Broad capabilities for editing configurations, but with some restrictions. 3️⃣ **RHEL Viewer**: Read-only access for viewing system data.
