AI Assistant, Junie and Kineto Now Support GPT-5 by OpenAI

2025-08-07 17:00

🚀 Exciting news from JetBrains! The latest model from OpenAI, GPT-5, is now integrated into AI Assistant, Junie, and Kineto. This update enhances coding capabilities, improving code quality and handling complex tasks more effectively. Junie, now powered by GPT-5, shows significant performance gains, making it easier to create production-ready code. To experience GPT-5, update to JetBrains IDE version 2025.2 and install the latest AI Assistant and Junie plugins. #JetBrains #GPT5 #AI #Coding…
