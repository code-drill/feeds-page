🚀 Exciting updates in Vercel Observability! The metrics search and navigation experience has been enhanced. You can now quickly find metrics using partial names or abbreviations like TTFB (time to first byte). Additional features include a side-by-side view of all metrics for an event, keyboard shortcuts for faster navigation, and a mobile-optimized interface. These updates are available now for all teams using Observability Plus. #Vercel #Observability #TechUpdates #Metrics #Productivity
