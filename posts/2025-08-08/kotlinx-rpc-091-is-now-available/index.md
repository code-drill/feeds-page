🚀 KotlinX RPC 0.9.1 has been released with key improvements for stability and user experience. The library now decouples from KotlinX Serialization, allowing for more flexible gRPC setups. Resource management has been simplified, reducing potential errors in implementation. Strict mode is now the default, ensuring better API correctness. Ready to dive in? Start your project at start.ktor.io and explore samples on GitHub! #KotlinXRPC #Kotlin #Programming #SoftwareDevelopment #OpenSource
