🚀 Exciting news for developers! Vercel now supports the vlt package manager for builds without the need for configuration. 📂 Projects with a vlt-lock.json file will automatically execute vlt install as the default Install Command. 🔧 Note: vlt requires Node 20.x and is available only in the modern build image. Learn more about this new feature on Vercel! #Vercel #vlt #WebDevelopment #PackageManager #TechNews
