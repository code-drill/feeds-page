🚀 Vercel has enhanced its bot verification system by supporting Web Bot Auth, developed through collaboration with industry partners. This new protocol uses HTTP Message Signatures for verifying traffic from various automated sources. It helps maintain a curated directory of verified bots, ensuring legitimate ones access your site while blocking spoofed bots.
