🚀 Exciting news for developers! Vercel has launched the Runtime Cache API, allowing you to store and retrieve data seamlessly across Functions, Routing Middleware, and Builds within the same region. This ephemeral cache offers tag-based invalidation for better cache management. You can monitor performance metrics like hit rates and storage usage in the Observability dashboard. Keep in mind, reads and writes are billed regionally based on your runtime area. Learn more in the documentation! 📊...
