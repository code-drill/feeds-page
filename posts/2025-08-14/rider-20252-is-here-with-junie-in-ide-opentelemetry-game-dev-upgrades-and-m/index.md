🚀 Rider 2025.2 has launched, enhancing development tools with new features. Meet Junie, the AI coding agent, now faster and more capable, offering high-level brainstorming and hands-on coding support. It also integrates with GitHub for efficient pull request management. The updated Monitoring tool window simplifies performance optimization, while the new OpenTelemetry plugin provides runtime metrics directly in your IDE.
