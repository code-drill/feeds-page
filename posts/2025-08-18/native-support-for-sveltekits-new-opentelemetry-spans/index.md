🚀 Vercel has added native support for SvelteKit's new server-side OpenTelemetry spans. To begin, users can activate experimental tracing in SvelteKit and set up a tracing instrumentation file with the Vercel OpenTelemetry collector. Generated traces will now include built-in SvelteKit spans, and other collectors can also be configured. For further details, check the SvelteKit observability documentation. 📊 #SvelteKit #OpenTelemetry #Vercel #WebDevelopment #Observability
