🚀 Exciting news for HCP Terraform and Terraform Enterprise users! You can now assign default agent pools at the project level. This update simplifies the management of self-hosted agents, enhancing security and scalability. With this feature, project owners can easily set a default agent pool for future workspaces, promoting consistency across projects. Existing workspaces maintain their current settings, ensuring a smooth transition. Get started today and streamline your infrastructure...
