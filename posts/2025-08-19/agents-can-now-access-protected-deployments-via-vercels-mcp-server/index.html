Agents can now access protected deployments via Vercel’s MCP server

2025-08-19 13:00

🚀 Exciting news for developers! Vercel has introduced two new tools in its MCP server. The get_access_to_vercel_url feature allows agents to generate a temporary shareable URL, enabling access to Vercel Auth-protected deployments without login credentials. Additionally, the web_fetch_vercel_url tool lets agents fetch content directly from protected deployments, even when access would normally be denied. Explore these features to streamline your development process! #Vercel…
