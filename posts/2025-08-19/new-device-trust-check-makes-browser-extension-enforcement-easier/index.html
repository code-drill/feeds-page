New Device Trust Check makes browser extension enforcement easier

2025-08-19 00:00

🔒 1Password has launched the new Device Trust Check to enhance security in managing browser extensions. This feature allows admins to ensure the 1Password browser extension remains installed and active on user devices. If the extension is missing, users receive prompts to restore it before access to the desktop app is denied. The integration between the Extended Device Compliance feature and EPM strengthens security enforcement continuously, closing gaps for potential compliance issues….
