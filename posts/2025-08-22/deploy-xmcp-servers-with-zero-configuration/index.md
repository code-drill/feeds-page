🚀 Vercel now supports xmcp, a framework for building MCP servers using TypeScript with zero configuration. 📁 It utilizes file-based routing, making it easy to develop tools for your MCP server. You can create tools simply by adding a file with a default export. 🛠️ Learn more about deploying xmcp to Vercel in the documentation. #Vercel #xmcp #TypeScript #WebDevelopment #MCPServers
