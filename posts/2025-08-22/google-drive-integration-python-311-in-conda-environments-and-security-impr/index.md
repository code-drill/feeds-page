🚀 Datalore 2025.4 is now live! Key updates include support for Google Drive folders as data connections, allowing seamless access to files like CSV and JSON without manual uploads. Python 3.11 is the new default in conda environments, enhancing performance and security. Users can also customize their Python versions as needed. Security improvements have been made to address CSRF and XSS vulnerabilities, ensuring safer usage. #Datalore #DataScience #Python #CloudStorage #SecurityUpdates
