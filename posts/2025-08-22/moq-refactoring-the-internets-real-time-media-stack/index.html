MoQ: Refactoring the Internet's real-time media stack

2025-08-22 14:00

🌐 Media over QUIC (MoQ) is a new IETF standard transforming real-time media streaming by unifying various protocols. For years, we relied on RTMP, HLS, and WebRTC, each addressing specific needs but creating complexity in the process. MoQ aims to streamline this by combining low latency with scalability and operational simplicity. Cloudflare has launched the first MoQ relay network, collaborating with industry leaders like Meta and Google to create a shared foundation for future applications….
