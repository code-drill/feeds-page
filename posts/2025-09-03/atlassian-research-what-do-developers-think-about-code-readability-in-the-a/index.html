Atlassian Research: What Do Developers Think About Code Readability in the Age of LLMs?

2025-09-03 16:20

Atlassian’s research explores the importance of code readability in the era of Large Language Models (LLMs) like RovoDev. Key findings show that 81% of developers still view code readability as essential, primarily for reducing maintenance costs. Despite LLMs writing more code, many practitioners are open to using these tools to enhance readability.
