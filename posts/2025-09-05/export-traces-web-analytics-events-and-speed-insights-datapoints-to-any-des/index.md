🚀 Exciting news for Vercel users! You can now export OpenTelemetry traces, Web Analytics events, and Speed Insights data to any third-party tool. This enhancement allows streaming of more raw data through expanded Log Drains infrastructure. Pro and Enterprise teams can set custom HTTP endpoints and choose from various encodings like JSON, NDJSON, or Protobuf. Explore Vercel Drains today! #Vercel #DataExport #WebAnalytics #OpenTelemetry #TechUpdates
